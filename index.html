Projects

MeetMe API Hack

A travel solutions app for bringing people together in a common destination, inspired by my own personal struggle to plan trips with long distance friends and family. MeetMe allows you to find flight information for two different departure cities to a common destination city, while also displaying photos of where you are headed.

Created with HTML, CSS, JavaScript and REST APIs.

Jot Note Keeping App

A clean and simple note taking app that is fully responsive. Allows users to easily add notes and folders and keep their thoughts organized.
Coming soon: server side with Node.js.

Created with React, JavaScript, and CSS Grid.

Harry Potter Quiz App

This project was inspired by my embarrassing knowledge of the Harry Potter franchise. I created an interactive quiz for fellow enthusiasts that allows the user to easily iterate through each question, ultimately getting their final score of correctly answered questions.

Created with HTML, CSS, JavaScript and jQuery.

What to Wear

A mobile-first CRUD app where users give and receive feedback on what should be worn to an occasion. Inspired by my nostalgia for getting constant outfit feedback while living with friends in college.
Currently in production. Final version coming soon!

Created with JavaScript, React, Node.js, Express, PostgreSQL and CSS Grid.
